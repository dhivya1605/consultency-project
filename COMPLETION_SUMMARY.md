## 🚀 How to Use

### Step 1: Start All 3 Servers

```bash
# Terminal 1: ML API (Port 8000)
cd ml_models
python api/app.py

# Terminal 2: Backend (Port 5000)
cd backend
npm run dev

# Terminal 3: Frontend (Port 3000)
cd frontend
npm start
```

### Step 2: Access Reports Page

1. Open browser: `http://localhost:3000`
2. Click "Reports" in navigation
3. Or go directly: `http://localhost:3000/reports`
4. Wait 5-10 seconds for all data to load
5. Scroll through all 20 analyses

### Step 3: Explore Analyses

- View charts and visualizations
- Check tables for detailed data
- Review cards for key metrics
- Use insights for business decisions

## 🚨 Troubleshooting

### Charts not loading?
1. Verify all 3 servers running
2. Check browser console (F12)
3. Hard refresh (Ctrl+Shift+R)
4. Check network tab

### Endpoints not working?
1. Verify backend on port 5000
2. Check backend logs
3. Restart backend server
4. Verify ML API on port 8000

### Slow loading?
1. First load is normal (5-10 seconds)
2. Check network tab
3. Verify dataset loaded
4. Check console errors
